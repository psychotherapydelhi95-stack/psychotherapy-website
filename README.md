# Psychotherapy Supervision Website

A professional, multi-page website designed for a Psychotherapy Supervisor. This static site features a calm, modern design tailored for the mental health profession.

## Features
*   **Multi-Page Architecture**: Dedicated pages for Home, About, Services, and Contact.
*   **Responsive Design**: Fully mobile-friendly layout with a custom hamburger menu.
*   **SEO Optimized**: Includes semantic HTML and Schema.org `ProfessionalService` markup.
*   **Premium Esthetics**: Uses a custom "Deep Teal & Sage" color palette with elegant typography.

## Project Structure
*   `index.html`: The main landing page.
*   `about.html`: Supervisor biography and philosophy.
*   `services.html`: Details on Individual, Group, and Online supervision.
*   `contact.html`: Contact form and information.
*   `style.css`: All custom styling (no frameworks used).
*   `script.js`: Handles mobile navigation and smooth scrolling.

## How to Run Locally
1.  Download the project files.
2.  Open `index.html` in any web browser (Chrome, Safari, Firefox).
3.  Navigate through the links to verify all pages work correctly.

## Deployment Guide (GitHub + Vercel)
This project is optimized for modern deployment workflows.

### 1. Upload to GitHub
1.  Create a new repository on [GitHub](https://github.com).
2.  Upload all files (`.html`, `.css`, `.js`, `.md`) to the repository main branch.

### 2. Connect to Vercel
1.  Log in to [Vercel](https://vercel.com) using your GitHub account.
2.  Click **Add New...** > **Project**.
3.  Import the `psychotherapy-website` repository.
4.  Keep default settings and click **Deploy**.
5.  Your site is now live!
